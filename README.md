## 0. 課程願景 (Course Vision)

本課程導入 **"Vibe Design"** 核心概念，主張 **80/20 法則**：AI 執行 80% 勞力，人類負責 20% 的決策。 為豐富作品多樣性，本課程採 **「雙軌主題制」**，每堂課提供兩種截然不同的案型供各組選擇。

## 1. 課程基本資訊

- **總時數**：36 小時 (12 堂)
- **班級人數**：25~30 人 (6 組)
- **主題選擇機制**：每堂課開始時，各組可自由選擇 **【主題 A】** 或 **【主題 B】** 進行實作。
    - **主題 A (Future Tech)**：偏向科技、電競、新創、高對比風格。
    - **主題 B (Slow Life)**：偏向飲食、手作、旅遊、自然風格。
- **工具矩陣**：Microsoft Designer (Bing), Ideogram, Recraft.ai, Canva, Affinity, Gemini.

## 2. 分組角色與職責 (Role Play)

*(與原版相同，強調角色分工)*

1. **專案總監 (Vibe Keeper)**：決定本週要選主題 A 還是 B，並控管風格。
2. **自然語言溝通師 (Chat Designer)**：根據選定主題修改 Prompt。
3. **素材獵人 (Assets Hunter)**：生成對應的局部素材。
4. **視覺組裝師 (Visual Assembler)**：Affinity/Canva 操刀手。
5. **商業敘事者 (Storyteller)**：提案簡報。

## 4. 每堂課流程規畫 (3 小時結構)

| **時間**            | **內容**                      | **活動詳情**                                                                  |
| ----------------- | --------------------------- | ------------------------------------------------------------------------- |
| **00:00 - 00:30** | **Vibe 設定與自然語言示範**          | 講師 Live Demo：如何用「聊天」修正 AI 的錯誤。展示 **Affinity** 如何修補 AI 做不到的細節（如手指修正、邊緣去背）。 |
| **00:30 - 01:00** | **資源盤點與腦力激盪**               | 各組確認今日可用額度（如：Bing 點數）。決定今天的風格關鍵字 (Vibe Keywords)。                         |
| **01:00 - 02:20** | **生成與組裝 (Hybrid Workflow)** | **關鍵環節**：AI 產出 80% 素材 -> 丟入 Affinity 進行 20% 的精細組裝（去背、調色、排版、向量化）。          |
| **02:20 - 02:30** | **除錯與品質控管**                 | 檢查印刷風險（如字體太小、圖片模糊、去背不乾淨），進行專業修正。                                          |
| **02:30 - 03:00** | **設計策略發表**                  | 重點：**「我們如何用免費工具解決商業問題？」** 分享與 AI 的對話過程與 Prompt 紀錄。                        |

## 5. 學生必備技能與心態 (Vibe Coding 版)

- **Chat Fluency (對話流利度)**：不怕跟機器人講話，懂得換個說法 (Rephrase) 來達到目的。
- **Hybrid Capability (雙刀流能力)**：具備判斷力，懂得何時該用 AI 偷懶 (Canva)，何時該用 Affinity 展現專業。
- **Asset Thinking (素材思維)**：不執著於一鍵生成完美作品，而是具備「拼貼」與「組裝」的能力。
- **Professional Eye (專業眼光)**：透過 Affinity 的操作，養成對細節（邊緣、對齊、色準）的敏銳度。

## 6. 講師備課重點 (Instructor Notes)

- **引導選題**：建議第一堂課就讓各組決定這學期的大方向是「科技派」還是「生活派」，這樣期末作品集會更有連貫性。但若為了練習不同風格，每週切換也是允許的。
- **風格對照表**：上課時，講師可準備「同一種東西（如咖啡）」在兩種風格下的不同 Prompt 結果（一杯在霓虹燈下的濃縮咖啡 vs. 一杯在窗邊的手沖咖啡），讓學生秒懂 Prompt 對氛圍的影響。
