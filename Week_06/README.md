# 第 6 堂：懶人包與圖表 - 讓 AI 畫圖標

- **主題**：資訊圖表與知識型內容
- **工具**：Recraft.ai (Icon 生成) + Napkin.ai + Affinity
- **重點方向**：
    - **Icon Set 生成**：用 Recraft 產出整套風格統一的 SVG 圖標。
    - **SVG 深度編輯**：將 SVG 匯入 Affinity Designer 修改節點與配色（Canva 僅能改色，無法改形狀）。
    - **Napkin.ai 實戰**：將文字一鍵轉化為流程圖結構。
- **實作產出**：社群知識型懶人包 (Infographic)。
