# 第 6 堂：懶人包與圖表 - 讓 AI 畫圖標

- **主題**：資訊視覺化與圖標設計
- **工具**：AI Icon 生成 + Affinity Designer (向量化)
- **重點方向**：
    - **資訊簡化**：將複雜的文字流程轉化為直觀的圖解。
    - **風格統一**：確保所有 Icon 的線條粗細、圓角與配色一致。
    - **模組化設計**：建立可重複使用的圖表元件庫。
- **實作產出**：一份 4-6 頁的知識型懶人包 (Carousel)。

## 雙軌實戰任務 (Dual-Track Specs)
#### **第 6 堂：懶人包與圖表 - 讓 AI 畫圖標**

- **學習主軸**：AI 生成圖標素材 + Affinity 資訊排版。
- **【主題 A：區塊鏈/AI 運作原理】**
    - **風格**：**等距視角 (Isometric)** - 2.5D、科技感、精細。
    - **內容**：如何鑄造 NFT / AI 生成流程。
    - **參考文案內容 (Reference Copy)**：
        - **標題**：NFT 鑄造流程全解析
        - **步驟一**：準備數位資產 (Artwork)
        - **步驟二**：連接加密錢包 (Wallet)
        - **步驟三**：上傳至交易平台 (Minting)
        - **步驟四**：設定價格與版稅 (Listing)
    - **Prompt 進階三部曲 (由淺入深)**：
        1. **Lv1 (基礎)**：`Computer icon, isometric. (電腦圖標，等距視角)`
        2. **Lv2 (中階)**：`Blockchain technology icon, isometric style, 3d render, blue and white. (區塊鏈科技圖標，等距視角風格，3D 渲染，藍與白)`
        3. **Lv3 (高階)**：`Isometric illustration of a futuristic server room, connecting nodes, data flow, soft lighting, 3d render, blender style, white background, isolated elements for infographic. (未來伺服器機房等距視角插圖，連接節點，數據流，柔和燈光，3D 渲染，Blender 風格，白底，獨立元素適合資訊圖表)`

    - **內容**：手沖咖啡步驟 / 龜背芋照顧指南。
    - **參考文案內容 (Reference Copy)**：
        - **標題**：完美手沖咖啡指南
