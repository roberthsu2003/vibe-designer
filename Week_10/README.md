# 第 10 堂：大型輸出 - 解析度救星

- **主題**：易拉展與戶外看板
- **工具**：免費 AI Upscaler + Affinity (大圖排版)
- **重點方向**：
    - **解析度焦慮**：解決 AI 生成圖 (通常僅 1024px) 無法印刷大圖的問題。
    - **DPI 設定**：理解螢幕 (72dpi) 與印刷 (150-300dpi) 的差異。
    - **CMYK 色差控管**：在 Affinity 開啟軟樣打 (Soft Proofing) 預覽 CMYK 印刷效果，避免色偏。
- **實作產出**：1:1 真實尺寸的易拉展設計稿。
