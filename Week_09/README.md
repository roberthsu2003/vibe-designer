# 第 9 堂：包裝模擬 - 3D Mockup 與刀模

- **主題**：產品包裝視覺設計
- **工具**：Leonardo.ai (材質紋理) + Affinity (刀模繪製) + Canva (Mockup)
- **重點方向**：
    - **刀模圖認知**：在 Affinity 繪製標準的包裝展開圖。
    - **材質生成**：生成「牛皮紙」、「雷射膜」等特殊紋理作為底圖。
    - **虛實整合**：將平面設計匯出，利用 Canva 的 Smartmockups 進行 3D 預覽。
- **實作產出**：產品包裝展開圖 + 3D 模擬情境圖。
