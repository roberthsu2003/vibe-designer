# 第 12 堂：Vibe Design 成果發表會

- **主題**：Demo Day - 設計提案
- **流程**：
    - **對比展示**：展示「AI 原圖」vs.「Affinity 完稿圖」，凸顯人類加工的價值。
    - **專業度證明**：展示 Affinity 的圖層結構，證明這不是「一鍵生成」的運氣，而是「精密組裝」的實力。
    - **互評環節**：評選最佳「自然語言溝通師」與最佳「視覺總監」。

## 4. 每堂課流程規畫 (3 小時結構)

| **時間**            | **內容**                      | **活動詳情**                                                                  |
| ----------------- | --------------------------- | ------------------------------------------------------------------------- |
| **00:00 - 00:30** | **Vibe 設定與自然語言示範**          | 講師 Live Demo：如何用「聊天」修正 AI 的錯誤。展示 **Affinity** 如何修補 AI 做不到的細節（如手指修正、邊緣去背）。 |
| **00:30 - 01:00** | **資源盤點與腦力激盪**               | 各組確認今日可用額度（如：Bing 點數）。決定今天的風格關鍵字 (Vibe Keywords)。                         |
| **01:00 - 02:20** | **生成與組裝 (Hybrid Workflow)** | **關鍵環節**：AI 產出 80% 素材 -> 丟入 Affinity 進行 20% 的精細組裝（去背、調色、排版、向量化）。          |
| **02:20 - 02:30** | **除錯與品質控管**                 | 檢查印刷風險（如字體太小、圖片模糊、去背不乾淨），進行專業修正。                                          |
| **02:30 - 03:00** | **設計策略發表**                  | 重點：**「我們如何用免費工具解決商業問題？」** 分享與 AI 的對話過程與 Prompt 紀錄。                        |

## 5. 學生必備技能與心態 (Vibe Coding 版)

- **Chat Fluency (對話流利度)**：不怕跟機器人講話，懂得換個說法 (Rephrase) 來達到目的。
- **Hybrid Capability (雙刀流能力)**：具備判斷力，懂得何時該用 AI 偷懶 (Canva)，何時該用 Affinity 展現專業。
- **Asset Thinking (素材思維)**：不執著於一鍵生成完美作品，而是具備「拼貼」與「組裝」的能力。
- **Professional Eye (專業眼光)**：透過 Affinity 的操作，養成對細節（邊緣、對齊、色準）的敏銳度。
