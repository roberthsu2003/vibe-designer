# 第 7 堂：品牌 Logo - 向量化實戰 (Affinity 主場)

- **主題**：Logo 設計與向量繪圖
- **工具**：AI 靈感生成 + Affinity Designer (鋼筆工具)
- **重點方向**：
    - **向量優勢**：理解點陣圖 (Raster) 與向量圖 (Vector) 的差異，以及 Logo 為何必須是向量。
    - **鋼筆工具**：掌握貝茲曲線 (Bézier Curve) 的繪製技巧，精準描繪 AI 生成的草圖。
    - **黃金比例**：運用幾何圖形輔助 Logo 構圖，提升視覺平衡感。
- **實作產出**：一組標準化 Logo (包含黑白稿、標準色稿、反白稿)。

## 雙軌實戰任務 (Dual-Track Specs)
#### **第 7 堂：品牌 Logo - 向量化實戰**

- **學習主軸**：AI 生成 Logo 草圖 + Affinity 向量描繪。
- **【主題 A：科技新創/SaaS 平台】**
    - **風格**：**幾何漸層 (Geometric Gradient)** - 簡約、現代、速度感。
    - **品牌名**：NEXUS / CORE
    - **參考文案內容 (Reference Copy)**：
        - **品牌名稱**：NEXUS
        - **Slogan**：Connecting the Future
        - **品牌理念**：連結無限可能，創造智慧生活。
    - **Prompt 進階三部曲 (由淺入深)**：
        1. **Lv1 (基礎)**：`Logo for tech company, letter N. (科技公司 Logo，字母 N)`
        2. **Lv2 (中階)**：`Minimalist logo design, letter N, geometric shapes, blue gradient, white background. (極簡 Logo 設計，字母 N，幾何形狀，藍色漸層，白底)`
        3. **Lv3 (高階)**：`Modern tech startup logo, abstract letter N formed by connecting nodes, gradient blue to purple, vector style, flat design, white background, dribbble style, simple and scalable. (現代科技新創 Logo，由連接節點組成的抽象字母 N，藍到紫漸層，向量風格，扁平設計，白底，Dribbble 風格，簡約且可縮放)`
