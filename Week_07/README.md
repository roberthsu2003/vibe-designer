# 第 7 堂：品牌 Logo - 向量化實戰 (Affinity 主場)

- **主題**：品牌識別系統 (CIS) 初探
- **工具**：Ideogram (發想) + Vectorizer.ai + Affinity Designer
- **重點方向**：
    - **向量的必要性**：Logo 必須是向量檔 (Vector) 才能應用於各種尺寸而不失真。
    - **向量工作流**：AI 點陣草圖 -> Vectorizer 轉檔 -> Affinity 修整節點 (Node Tool)。
    - **標準字設計**：在 Affinity 中繪製專屬品牌字體。
- **實作產出**：品牌 Logo 原始檔 (SVG) 與色彩規範手冊。
