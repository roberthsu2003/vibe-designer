# 第 4 堂：菜單設計 - 視覺味蕾行銷

- **主題**：美食照片生成與排版
- **工具**：Microsoft Designer (生食物) + Affinity Publisher/Designer
- **重點方向**：
    - **美味的形容詞**：學習描述食物的「光澤感」、「熱氣」、「垂涎欲滴」。
    - **素材拆解法**：AI 畫不好整桌菜？那就一次畫一道，再進行組合。
    - **Affinity 排版強項**：學習對齊工具、引導線與網格系統，製作整齊易讀的價目表。
- **實作產出**：A4 單頁式餐廳菜單。
