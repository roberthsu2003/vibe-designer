# 第 11 堂：整合行銷戰役 (Capstone Project)

- **主題**：全通路行銷素材大亂鬥
- **重點方向**：
    - **One Vibe, Multi-Format**：確立一個主視覺，延伸至不同載體。
    - **高效流水線**：
        - A/B 同學：利用 Gemini/Bing 快速生成大量素材。
        - C 同學：利用 Affinity 處理高解析度主視覺與 Logo。
        - D 同學：利用 Canva 快速量產 IG 限動與 Banner。
- **實作產出**：包含 3 種不同尺寸 (海報、Banner、限動) 的行銷素材包。
