- **主題**：期末專案衝刺與整合
- **工具**：全工具矩陣 (All Tools)
- **重點方向**：
    - **專案管理**：學習如何規劃一個完整的設計專案，從發想到執行。
    - **風格統一**：確保 Logo、海報、社群貼文、包裝等所有素材的視覺語彙一致。
    - **Mockup 展示**：製作高品質的 Mockup 來展示設計成果，提升提案說服力。
- **實作產出**：期末專案初稿 (包含至少 3 種不同類型的設計產出)。

- **【主題 A：AI 應用程式/科技新品發布】**
    - **專案目標**：為一個虛構的 AI App 或科技產品設計全套品牌識別與行銷素材。
    - **必要產出**：
        1. **Logo** (Week 7 技巧)
        2. **App Store 預覽圖/網站首圖** (Week 3/5 技巧)
        3. **社群推廣貼文** (Week 2 技巧)

        - **功能亮點**：AI 智慧摘要、語音轉文字、心智圖自動生成。
        - **Call to Action**：Download Now on App Store.

    - **Prompt 建議 (全方位結構)**：
